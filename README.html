Webconverger daily builds

Scared? Please see the Webconverger download site for tried and tested end user releases.

Filenames should correspond to `git describe`'s tags and commits.

  • http://build.webconverger.org/latest.iso

Hints on testing can be found on the wiki.

Please report any bugs or issues to the mailing list or issue tracker. Thank you!

Source code on github of this build tool

Cronjob

As root:

@daily /srv/www/build.webconverger.org/webc-build.sh
@daily find /srv/www/build.webconverger.org/ -mindepth 1 -maxdepth 1 -mtime +4 \( -name "*.txt" -o -name "*.iso" \) -exec rm -vrf {} \;
